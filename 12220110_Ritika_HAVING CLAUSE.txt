============================
	HAVING CLAUSE
============================

1 - WAQTD DNO AND NUMBER OF EMP WORKING IN EACH DEPT IF THERE ARE ATLEAST 2 CLERKS IN EACH DEPT.
SQL> SELECT deptno, COUNT(*) FROM emp WHERE job='CLERK' GROUP BY deptno HAVING COUNT(*)>=2;

2 - WAQTD DNO AND TOTAL SALARY NEEDED TO PAY ALL EMP IN EACH DEPT IF THERE ARE ATLEAST 4 EMP IN EACH DEPT. 
SQL> SELECT deptno, SUM(sal) FROM emp GROUP BY deptno HAVING COUNT(*)>=4;

3 - WAQTD NUMBER OF EMP EARNING SAL MORE THAN 1200 IN EACH JOB AND TOTAL SAL OF EACH JOB MUST EXCEED 3800. 
SQL> SELECT job, COUNT(*), SUM(sal) FROM emp WHERE sal>1200 GROUP BY job HAVING SUM(sal)>3800;

4 - WAQTD DEPTNO AND NUMBER OF EMP WORKING ONLY IF THERE ARE 2 EMP WORKING IN EACH DEPT AS MANAGER. 
SQL> SELECT deptno, COUNT(*) FROM emp WHERE job='MANAGER' GROUP BY deptno HAVING COUNT(*)=2;

5 - WAQTD JOB AND MAX SAL OF EMP IN EACH JOB IF THE MAX SAL EXCEEDS 2600. 
SQL> SELECT job, MAX(sal) FROM emp GROUP BY job HAVING MAX(sal)>2600;

6 - WAQTD THE SALARIES WHICH ARE REPEATED IN EMP TABLE. 
SQL> SELECT sal, COUNT(*) FROM emp GROUP BY sal HAVING COUNT(*)>1;

7 - WAQTD THE HIREDATE WHICH ARE DUPLICATED IN EMP TABLE. 
SQL> SELECT hiredate, COUNT(*) FROM emp GROUP BY hiredate HAVING COUNT(*)>1;

8 - WAQTD AVG SALARY OF EACH DEPT IF AVG SAL IS LESS THAN 3000. 
SQL> SELECT deptno, AVG(sal) FROM emp GROUP BY deptno HAVING AVG(sal)<3000;

9 - WAQTD DEPTNO IF THERE ARE ATLEAST 3 EMP IN EACH DEPT WHOSE NAME HAS CHAR 'A' OR 'S'. 
SQL> SELECT deptno, COUNT(*) FROM emp WHERE ename LIKE '%A%' OR ename LIKE '%S%' GROUP BY deptno HAVING COUNT(*)>=3;

10 - WAQTD MIN AND MAX SALARIES OF EACH JOB IF MIN SAL IS MORE THAN 1000 AND MAX SAL IS LESS THAN 5000. 
SQL> SELECT job, MIN(sal), MAX(sal) FROM emp GROUP BY job HAVING MIN(sal)>1000 AND MAX(sal)<5000;
